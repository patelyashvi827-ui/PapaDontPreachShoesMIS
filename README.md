# 🐾 Papa Don't Preach - MIS Tracker

A premium, real-time Management Information System (MIS) designed for tracking production and sampling workflows at **Papa Don't Preach by Shubhika**.

## 🚀 Features
- **Cloud Sync**: Powered by Firebase for real-time data across all devices.
- **Role-Based Access**: Specialized dashboards for Admins (Managers) and Executives.
- **Production Pipeline**: Track orders from Procurement through Finishing and QC.
- **Sampling Dashboard**: Monitor design development and production readiness.
- **Automated Calculations**: Instant selling price and order value logic based on MRP and commissions.
- **Smart Alerts**: Automatic notifications for dispatch delays.

## 🛠️ Setup Instructions
1. **Clone/Download**: Download this repository to your computer.
2. **Firebase Config**:
   - Open `index.html`.
   - Update the `firebaseConfig` object (line 1127) with your keys from the [Firebase Console](https://console.firebase.go.com/).
3. **Deployment**:
   - Upload this folder to a GitHub repository.
   - Go to **Settings > Pages** and enable GitHub Pages on the `main` branch.
   - Your tracker will be live at `https://yourusername.github.io/PDP-MIS/`.
